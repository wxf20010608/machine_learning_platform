## 🚀 使用方法

### 快速启动
```bash
# 方法1: 使用启动脚本（推荐）
python start_docs.py

# 方法2: 手动启动
python main.py
# 然后访问 http://localhost:8000/docs
```

### 生成静态文档
```bash
python generate_docs.py
```

### 访问文档
- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc
- **静态HTML**: `docs/API.html`
- **Markdown**: `docs/API.md`
